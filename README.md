### 🤚 Commit Message HEAD Convention


- feat: 기존에 없던 기능 개발

- ADD : 부가적인 파일 추가

- fix: 버그, 이슈 수정

- docs: 문서나 첨부 링크 등의 추가 및 수정

- style: 코드 컨벤션 등의 수정 

- design: layout,style 등의 수정

- edit: 코드 변경 사항 수정(주로 PR 요청 전의 간단한 수정)

- refactor: 코드를 리팩토링(팀원리뷰/자가리뷰 후 개선)

- test: 테스트 코드 추가

- chore: 패키지 설치 및 환경 설정에 관련된 내용이 수정
